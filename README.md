# GO Boilerplate

GO Boilerplate

## Installation

Install [Docker](https://docs.docker.com/engine/install/ubuntu/)

Install [Docker Compose](https://docs.docker.com/compose/install/)

```bash
docker-compose up -d
```

## Usage

```bash
docker exec -it auth bash
```

```bash
go run cmd/auth/main.go
```
